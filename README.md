# Vehicle Test App

ASP.NET Core MVC application:
- Entity Framework Core
- CRUD operations
- Sorting, filtering and paging
- Dependency Injection
- AutoMapper

# Entities
- VehicleMake (Id, Name, Abrv)
- VehicleModel (Id, MakeId, Name, Abrv)

# Features
- Make administration (CRUD, sorting, filtering, paging)
- Model administration (CRUD, sorting, filtering, paging)
- Filtering models by make

# Technologies
- ASP.NET Core MVC
- Entity Framework Core
- AutoMapper
- SQL Server LocalDB
